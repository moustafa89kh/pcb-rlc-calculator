RLC-calculator for PCB Trace

By Aleksandr Sidun

Overview

This article contains a MATLAB script for calculating the parasitic parameters of a PCB trace.

Script Limitations

  • Valid under 1 GHz
  • Skin effect is omitted
  • Copper roughness is omitted
  • Loss tangent is omitted

Geometry Limitations

  • W/H < 7.475 - 1.25 * (T/H) for a microstrip
  • W/B < 2.375 - 1.25 * (T/B) for a stripline

Formulas

Microstrip Equations:

        R = 1000 * ρ₀ * (1 + α * (temp - 25)) / (T * W)
        C = 26.378 * (ϵr + 1.41) * ln(5.98 * H / (0.8 * W + T))
        L = 199.65 * ln(5.98 * H / (0.8 * W + T))
        Z = 87 * ln(5.98 * H / (0.8 * W + T)) / sqrt(ϵr + 1.41)

Stripline Equations:

        R = 1000 * ρ₀ * (1 + α * (temp - 25)) / (T * W)
        C = 39.37 * ϵr * sqrt(2) / (ln(1.9 * B / (0.8 * W + T)))
        L = 199.8425 * ln(1.9 * B / (0.8 * W + T))
        Z = 60 * ln(1.9 * B / (0.8 * W + T)) / sqrt(ϵr)

PCB Lumped Parameters Calculator

Use the MATLAB script to calculate R, L, C, and Z for your PCB traces (microstrip or stripline).
